## Type
  ชนิดของตัวแปรใน Javascript นั้น จากที่ผมได้พัฒนา Web Applicationได้เจออะไรแปลก ๆ มาพอสมควร งั้นผมจะแจกแจงที่ละตัวเลยนะครับ
  1. [string]() (ตัวอักษร)
  
  > เอาไว้เก็บข้อมูลจำพวก ข้อความ, ตัวอักษร ใน Javascript นั้น การกำหนดข้อความต้องทำสัญลักษณ์เปิดและปิด ทำได้ 2 แบบ คือ          1. single quote(') ตัวอย่าง "Heng Indy Anvil", "123456", "♥", "<html></html>                                                   2. double quote(") ตัวอย่าง 'Heng Indy Anvil', '123465', '♥', '<html></html>
  1. [number]() (ตัวเลข)
  
  > เอาไว้เก็บข้อมูลจำพวก ตัวเลข ทั้งหมด ทั้งจำนวนเต็ม จำนวนจริง ตัวอย่าง 11, 11.23, -0.254564
  1. [boolean]() (เท็จจริง)
  
  > เอาไว้เก็บข้อมูลจำพวก ค่าเท็จจริง ตัวอย่าง true, false
